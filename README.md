# 🏅 Olympics Countdown Timer

A simple countdown timer built using HTML, CSS, and JavaScript.

## 🚀 Features

- Live countdown timer
- Displays Days, Hours, Minutes, and Seconds
- Updates every second
- Responsive design
- Beautiful gradient background

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
